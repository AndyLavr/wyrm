Wyrm

Wyrm is a small library of Bourbon powered SASS files meant for use in minimalistic Document and Form heavy sites. It was primarily built for Webhook but can also be found in use on Read the Docs sphinx theme. Wyrm is most similar to purecss in that it is not meant to exist as a all-inclusive framework like bootstrap but simply be a starting point for larger sass/bourbon projects.
