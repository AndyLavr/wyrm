1.0 - Buttons

Located in wyrm_core/_button.sass. You can also add font icons to any button as shown in the third example. Wyrm by default uses the FontAwesome library, but any icon set can be substituted.

<a href="#" class="btn {$modifiers}"> Link</a>
<button class="btn {$modifiers}"> Button</button>
<input type="submit" class="btn {$modifiers}">
<button class="btn fa fa-check-circle {$modifiers}"> With icon</button>
